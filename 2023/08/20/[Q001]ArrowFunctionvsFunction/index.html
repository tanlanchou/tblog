01. () => 有没有原型链

这个我居然真不知道，真没有注意到这个问题

首先我知道箭头函数是直接绑定父级别 this 的, 也就类似于

但是他确实没原型链

因为this指向上层的原因, 所以他不能作为构造函数

  • 箭头函数没有自己的 this，而是继承父作用域的 this
  • 不支持 call bind，改变 this 指向。
  • 不绑定 arguments
  • 没有 prototype 属性
